# my dotfiles

## Installtion
install zsh, curl, git

automatically:
```bash
$bash -c "$(curl -L raw.github.com/mizoc/dotfiles/master/install.sh)"
```

manually
```bash
$git clone https://github.com/mizoc/dotfiles ~/.dotfiles
$cd $_
$make
```

## Update
```bash
$dot update
```

## Check
```bash
$dot check
```

## Uninstalltion
```bash
$dot uninstall
```
