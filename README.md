# my dotfiles

## Installtion
`$cd`  
`$git clone https://github.com/mizoc/dotfiles ~/.dotfiles`  
`$cd $_`  
`$./install.sh`  
Please install zsh and excute  
`$chsh -s /bin/zsh`  

## Update
`$cd ~/.dotfiles`  
`$./update.sh`
