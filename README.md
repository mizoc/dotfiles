# my dotfiles

## Installtion
`$git clone https://github.com/mizoc/dotfiles ~/.dotfiles`  
`$cd $_`  
`$make`  
Please install zsh and excute  
`$chsh -s /bin/zsh`  

## Update  
`$cd ~/.dotfiles`  
`$./update.sh`  

## Uninstalltion  
`$cd ~/.dotfiles`  
`$make uninstall`
